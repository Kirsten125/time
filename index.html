GitHub Pages 部署方式

1. 建立一個新的 GitHub repository。
2. 把這份檔案命名為 index.html 上傳。
3. 到 GitHub 的 Settings → Pages。
4. Source 選 Deploy from a branch。
5. Branch 選 main，資料夾選 /root。
6. 儲存後，幾分鐘內就會拿到網站網址。
